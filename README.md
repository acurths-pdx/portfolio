# Alicia Curths – Portfolio

A personal portfolio website built with HTML, CSS, JavaScript, and Bootstrap 5. The site highlights my professional background, technical skills, and selected projects.

**Deployed site:** https://acurths-pdx.github.io/portfolio/

**GitHub repository:** https://github.com/acurths-pdx/portfolio

---

## Pages

- `index.html` — Main page with About, Experience, Projects, and Contact sections
- `contact.html` — Contact form page

## How to Run

No build step is required. Open `index.html` directly in a browser, or serve the directory with any static file server

## Outside Libraries and Sources

| Resource | Used for |

| [Bootstrap 5.3.3](https://getbootstrap.com/docs) | Page layout, responsive grid, navbar, accordion, carousel, modal, and utility classes |
